
				S&P500 Map

Introduction
	This project is a Map showing HQ location of all S&P 500 companies. Layers that allow you to filter by sector. Dropdown that will zoom you to a certain state/region (e.g. Silicon Valley). Click on a company and a sidebar will show basic company info, plus the current price/other relevant financial information.









List of sources used 

	1. List of S&P500 companies via CSV
		https://datahub.io/core/s-and-p-500-companies#resource-constituents
	2. API for relevant financial information 
		DOCS: https://iextrading.com/developer/docs/#getting-started
		SITE: https://iexcloud.io/console
	3. Address Information
